This is my personal portfolio website — built with HTML, CSS, and JavaScript to showcase my work, experience, and contact info.
I focused on keeping it clean, interactive, and easy to update.

Unique points

Custom Light Blue Theme — my default color palette with calm gradients and soft highlights.

Theme Toggle — switch between light (blue) and dark (violet-cyan) mode smoothly.

Running text- Added animations for typing style effect as well as profile image.

Experience Section — simple, timeline-like layout listing my key roles and skills.

Projects Section — responsive cards with hover effects for details and links.

Download CV Button — linked to Google Drive with direct download

Fully Responsive — adapts cleanly to phones, tablets, and desktops.
